1、进入官网下载安装软件（[XMind思维导图 | 中文官方网站](https://www.xmind.cn/xmind8-pro/)）或者在文末链接下载![](/images/2022/02/solo-fetchupload-13524962216971110485-8d7f39d4.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

2、安装完成后，打开软件安装地址

```
C:\Program Files (x86)\XMind
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

3、将XMindCrack.jar移动至此地址的文件夹中4、打开配置文件XMind.ini并在最后一行添加（-javaagent:./XMindCrack.jar）

```
-javaagent:./XMindCrack.jar
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

![](/images/2022/02/solo-fetchupload-8520418290924220132-3e78f82d.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")注意：打开时选择“用记事本打开”即可，条件允许建议使用VScode打开，并且最好在关闭前手动保存。否则可能会出现难以预料的错误，如最后程序打不开等，若出现此类错误，卸载软件重新安装即可。如果提示无法保存请查看**“附1”**

5、转到系统盘存放Host文件的地址

```
C:\Windows\System32\drivers\etc
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

6、找到Host文件（这个文件没有后缀名，同**第四步** 打开方式即可）![](/images/2022/02/solo-fetchupload-5846390794289474440-e18a1322.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

7、在末尾添加（127.0.0.1 [www.xmind.net）](http://www.xmind.net%29/)

```
127.0.0.1 www.xmind.net
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

![](/images/2022/02/solo-fetchupload-15425557230720336071-a5ea20f8.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")8、保存关闭所有窗口后，打开程序

9、在【帮助】中选择倒数第二项【序列号】输入**文件中提供的序列号** ，注册完成![](/images/2022/02/solo-fetchupload-9300747328262397440-c2f869a4.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

附1：

若提示无法保存文件，则：

-   先**右键文件** ==>**属性** ==>**安全**
-   单机**“组或用户名”** 栏右下角的**编辑**

![](/images/2022/02/solo-fetchupload-2404336551153826423-1fddbda4.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")\* 在新弹出窗口中的**“组或用户名”** 栏目中选中**开头为Users** 的账户

![](/images/2022/02/solo-fetchupload-11153893478267077764-cd2b75f9.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")\* 在下方的**“Users的权限”** 中，勾选**“完全控制”**

-   **单机确定**

![](/images/2022/02/solo-fetchupload-5910491295538087347-101a46c7.png)![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")\* 关闭**属性窗口**

-   回到上文正文的步骤中，重新尝试保存文件即可，此时就不会弹出“权限不够”的弹窗了
