# 准备材料

1、GitHub项目文件：[GitHub项目地址](https://github.com/Walker-King-Z/Personal-trial-project-1)（或选择站长准备的[蓝奏云地址](https://wwi.lanzouw.com/iGFiyybcmpe)）

2、[开通腾讯云函数](https://cloud.tencent.com/product/scf)（目前每月有优惠包，仅需1元，[开通地址](https://cloud.tencent.com/act/pro/scf_pkg?from=15302 "优惠获取")）

3、准备Cron表达式，可以到网站自行生成：[点击跳转](https://cron.qqe2.com/ "Cron表达式在线生成")（如果失效可以[点击搜索](https://cn.bing.com/search?q=cron%E8%A1%A8%E8%BE%BE%E5%BC%8F%E7%94%9F%E6%88%90&cvid=c8b12196b50e4eec9051be8ed069f928&aqs=edge..69i57.6830j0j1&pglt=163&FORM=ANNTA1&PC=U531 "Bing搜索“Cron在线生成”")）

```
也可以使用我生成好了的（30 8 * * *）意为每天早8:30执行一次
```

# 操作步骤

1、开通云函数后进入控制台

2、进入“**函数服务**”，点击“**新建**”

![图片暂时无法访问，等待后续更新.png](/images/image/20220103/1641194674947377.png "1641194674947377.png")

3、按照以下信息填入

```
选择“自定义创建”
选择“事件函数”
函数名字任取
选择“代码部署”、环境选择Nodejs最新的版本
提交方法为“本地上传Zip包”
执行方法为“index.main”
然后下面点击上传，上传准备工作1中下载的项目文件压缩包
```

![图片暂时无法访问，等待后续更新.png](/images/image/20220103/1641194884660785.png "1641194884660785.png")

4、点开“**高级配置**”

```
环境配置中：内存设置为128MB（一般为默认值）
环境配置中：执行超过时间为900秒
```

![图片暂时无法访问，等待后续更新.png](/images/image/20220103/1641195339977581.png "1641195339977581.png")

5、点开“**触发器配置**”

```
选择“自定义配置”
选择“自定义触发周期”
输入（准备步骤3中生成的）Cron表达式（可以使用我生成好了的【30 8 * * *】。）
（！！我截图中的表达式后来会报错，所以别按我下面的照截图写）
```

随后**点击“完成”** ，完成创建

![图片暂时无法访问，等待后续更新.png](/images/image/20220103/1641196179541346.png "1641196179541346.png")

![图片暂时无法访问，等待后续更新.png](/images/image/20220103/1641196689157212.png "1641196689157212.png")

6、找到配置文件

```
点击“函数管理”
点击“函数代码”
点击“代码编辑”（一般默认就已经选中）
点击“在线编辑”（一般默认就已经选中）
展开“config”文件夹
点击“account”文件
```

![图片暂时无法访问，等待后续更新.png](/images/image/20220103/1641196961751914.png "1641196961751914.png")

7、编辑账号文件

```
第一行：自己的网易云绑定的手机号
第二行：自己手机号对应的明文密码（直接输密码）
#如果手机号为国内号码，直接填入号码即可，但如果是国外手机号，则必须加区域码，比如美国手机号+1xxxxxxx
#支持多帐号同时刷，就按照格式：第一行账号，第二行密码；第三行账号，第四行密码。以此类推
```

8、点击底部“部署”即可完成！
